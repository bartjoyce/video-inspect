VideoInspect
============

Video file format inspector

## Build

```
$ git clone --recursive https://github.com/bartjoyce/video-inspect
$ cd video-inspect
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## Dependencies

- [ddui](https://github.com/bartjoyce/ddui)
- [portaudio](https://github.com/PortAudio/portaudio)
